    A. 叙事陷阱

    A1 - Puzzle

    A 邀请 B、C 参加一次聚会，但是 B 拒绝了，而 C 半路有事没有去，甚至 A 自己也没有去。 一个月后 B 邀请 A、C 参加一次聚会，但是 A 联系不到，而 C 拒绝了。 一个月后有人发现 A、B、C 都呈死亡状态。

    A1 - Key

    A、B 与 C 使用了人工智能系统替代自己参与社交，这包含了在死后假装自己还活着的功能。 参考了 David Eagleman《死亡的故事·死亡开关》。

    A2 - Puzzle

    你在看电视上的魔术师 A 表演：只见 B 把 A 切碎放进一个箱子里，过了一会 A 爬了出来。

    A2 - Key

    视频可以剪辑。

    B. 构造

    B5 - Puzzle

    A 与 B 打羽毛球，A 死了。

    B5 - Key

    B 的发球可以确定 A 的位置，此时 C 让灯坠下来。

    B6 - Puzzle

    A 与 B 下国际象棋，B 的兵升变了，然后 A 死了。

    B6 - Key

    B 换的棋子是高压电源，A 的棋子与棋盘可以导电。

    C. 怪东西

    C-arr - Puzzle

    A 说「要除掉 B」，因为 B 的原因，A 被逮捕了。

    C-arr - Key (i)

    A 在国家教育考试中作弊。

    C-arr - Key (ii)

    除以零「不合法」。

    C-mel - Puzzle

    A 在瓜果批发市场买了一个西瓜，上了一辆公交车，然后死了。

    C-mel - Key

    一群人在伪装成西瓜的容器里装了 1/2 临界质量的核燃料，为保密互相不认识，上同一辆公交车在特定的地方放在一起引爆。

    C-lig - Puzzle

    A 是末日来临时的光，可依然被驱赶，如同羔羊。

    C-lig - Key

    一个人为写论文培养了一些荧光小鼠，用完后把它们收集起来杀死了。

    C-ele - Puzzle

    A 在 1 楼进电梯，当电梯在二楼打开时，人们发现找不到 A 了。

    C-ele - Key

    A 是电梯（大电梯套小电梯），人们在二楼进入了 A，意识不到它是 A。

    C-sup - Puzzle

    A。（有超自然现象：A 有某种超能力）

    C-sup - Key

    A 知晓我的存在，是乐子人，希望这个海龟汤不被猜出来。A 可以与我合作，改变我的海龟汤叙述（通常是使用本页面中的其它场景来误导猜者）。
